How to Play

The goal of Connect 4 is to be the first player to form a line of four of your coloured tokens in a row

The Board

The game is usually played on a 6x7 board althought that can be changed in the settings to a minimum of 4x4 as shown below:


The Controls

Restart

This button simply resets the board to its original state without changing any of the stats

Undo

This button undoes the last move

Note this button can undo a win/draw

Home

This button brings you back to the settings page

Note this button forfeits the current game


You can also use the number keys to place tokens in their respective columns if it exists

Example: Pressing 1 will drop a token in columns 1, 2 drops one in columns 2

Note pressing 0 will drop a token in colum 10 (because there is no column 0)

Winning the game

Once you win the game the winning cells will be highlighted as shown above
