How to Play

Connect 4 is a two player game where you take turns dropping colored tokens into a grid. The goal is to be the first player to form line (vertical, horizontal, or diagonal) of 4 tokens.

Game Rules:

  • Players take turns dropping their tokens into any of the available columns.
  • Tokens fall to the lowest available position in the selected column.
  • The first player to connect four of their tokens in a row (horizontally, vertically, or diagonally) wins.
  • If nobody has won and the board is full then the game ends in a draw.

Making Moves

On Desktop:

  • Click on any column to drop your token
  • Hover over a column to see a preview of where your token will land
  • Use number keys (1-9, 0) to drop tokens in the corresponding columns (on a board with 10 columns the zero key will place a token in the 10th column)

On Mobile:

  • Tap on any column to drop your token
  • Swipe across columns to see placement previews

Move History

The game keeps track of all moves made during the current game.

On Desktop:

  • Hover over a move in the history to see a preview of the board at that point
  • Click on a move to return the game to that state

On Mobile:

  • Tap and hold a move in the history to see a preview of the board at that point
  • Tap a move to return the game to that state

The Controls

Restart

This button resets the board to its original state

Undo

This button undoes the last move in case a player made a mistake

Note: This button will undo a win or draw result as well.

Home

This button brings you back to the settings page where you can change player names, colors, and board dimensions.

Note: Going back to the settings forfeits the current game.


Keyboard controls: Use number keys to place tokens in columns:

  • Press 1 to drop a token in column 1
  • Press 2 to drop a token in column 2
  • And so on...
  • Press 0 to drop a token in column 10 (if it exists)

Game Settings

Before starting a game you can customize:

  • Player Names: Enter custom names for each player
  • Player Colors: Choose custom colors for each player's tokens
  • Board Size: Set the number of rows (4-10) and columns (4-10)
  • Board Color: Change the background color of the game board

All settings and statistics are saved automatically in your browser.

Winning the Game

When a player connects four tokens in a row (horizontally, vertically, or diagonally), those tokens will be highlighted with an animation as shown above.

The game will update the following:

  • The "Current Turn" display will change to show the winner
  • The statistics panel will update to record the win
  • The Restart button will be highlighted to encourage starting a new game

Draw Game

If all positions on the board are filled without any player connecting four in a row, the game ends in a draw. In this case:

  • The "Current Turn" display will indicate a draw
  • The statistics panel will update to record the draw
  • The Restart button will be highlighted to encourage starting a new game
